# Global Cancer Statistics — Exploratory Data Analysis

## Overview
An exploratory data analysis of global cancer incidence and mortality patterns
using GLOBOCAN 2022 data published by the WHO Global Cancer Observatory.

## Key Findings
- **Lung cancer** has the highest global incidence (2.48M new cases)
- **Pancreatic cancer** is the deadliest with a 92.2% mortality rate
- **Prostate cancer** has the best 5-year survival rate at 99%
- **Asia** carries the highest regional cancer burden globally
- 5 cancers exceed 70% mortality rate: Lung, Liver, Pancreatic, Leukemia, Esophageal
